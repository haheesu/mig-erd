# 데이터 통합 솔루션 기획 정리

공유용 정적 HTML 문서 패키지입니다.

포함 파일

- `index.html`: 문서 진입 페이지
- `phase1-connection-scope.html`: 1차 개발 범위 정리
- `mysql-settings-model.html`: MySQL DDL 및 설정정보 모델링
- `backend-api-spec.html`: 백엔드 명세 및 API endpoint 설계
- `assets/ppt-thumbnail.png`: 원본 기획서 썸네일 이미지

열기 방법

- 브라우저에서 `index.html` 파일을 열면 됩니다.
- 모든 리소스는 현재 폴더 내부 상대경로만 사용합니다.

공유 방법

- 현재 폴더 전체를 압축해서 전달하면 됩니다.
